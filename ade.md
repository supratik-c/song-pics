# Instructions

## Publishing the Website
1. Upload your images into a folder with the right date name (e.g. `2026-04-27`)
2. Make the puzzle.json file with the right info
3. Go back to the root folder (song-pics)
4. Right click and click `Show more Options` and click `Open Git Bash Here`
4. Type `git add . && git commit -m "Added new picture" && git push origin main` and hit Enter.
